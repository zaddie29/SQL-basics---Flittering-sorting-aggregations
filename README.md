# SQL-basics---Flittering-sorting-aggregations

This project is about practicing the basic SQL queries like flittering, sorting, and aggregation.

Dataset:
Superstore Sales Dataset

Querries used:
- WHERE: to filter data
- ORDER BY: to sort data
- GROUP BY + SUM/ AVG/ COUNT: to make summary
- HAVING: to filter grouped data
- BATWEEN: for date range
- LIKE: for name search

OUTPUT:
category_summary.csv contains:
- total sales per category
- average profit per category
- total orders per category
